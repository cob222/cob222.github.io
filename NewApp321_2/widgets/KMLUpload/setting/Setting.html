Type Attribute

Provides a default value to a specified attribute that determines the symbology for a given layer. If you have symbology set based on a specific field in your target feature layers, this will provide it with a value so that newly added features using this tool appear correctly on the map. Note: if the attribute/value does not exist, the tool will default to using the first option for each coded value field on the feature layer.
